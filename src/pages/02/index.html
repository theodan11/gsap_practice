Experimenting with Properties

We can animate with css properties yay!... But css properties must be used in camelCase 

                    
gsap.to(".box1", {
    opacity: 1,
    scale: 1,
    duration: 2,
    y: 100,
    x: 300,
    borderRadius: "100%",
    backgroundColor: "#4657bbff",
    yoyo: true,
    repeat: -1,
    ease: "bounce.out"
})

gsap.to(".box2",{
    opacity: 1,
    scale: 1.2,
    x: -400, 
    y: 20, 
    ease: "power2.out",
    yoyo: true, 
    repeat: 3
})
